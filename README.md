# E-Commerce Sales Analysis and Business Insights with SQL

## Project Overview

This project analyzes an e-commerce sales dataset using SQL to extract meaningful business insights. The analysis focuses on understanding product performance, brand contribution, customer feedback, platform performance, and sales trends over time.

The goal of this project is to demonstrate practical SQL skills used in real-world data analysis, such as data aggregation, ranking, window functions, and time-based analysis.

The dataset used in this project contains 10,000 transaction records**, representing orders placed on an e-commerce platform.

---

## Dataset Description

The dataset represents e-commerce transaction data containing information about products, brands, order details, customer feedback, and sales information.

Each row in the dataset represents a **single product order transaction**.

The dataset contains **10,000 records** and the following columns:

OrderID – Unique identifier for each order
Product – Name of the product purchased
Category – Product category
Brand – Brand name of the product
Platform – E-commerce platform where the product was sold
City – City where the order was placed
Price – Price of the product
Quantity – Number of items purchased in the order
TotalAmount – Total value of the order
Rating – Customer rating for the product
Reviews – Number of reviews received by the product
OrderDate – Date when the order was placed

This dataset enables analysis of:

* Product sales performance
* Category-level revenue distribution
* Brand contribution to revenue
* Platform performance
* Customer feedback through ratings and reviews
* Sales patterns across time

## Database Setup

### Create Database

CREATE DATABASE ecommerce_db;

### Use Database

USE ecommerce_db;

### Create Table

CREATE TABLE ecommerce (
OrderID VARCHAR(20),
Product VARCHAR(100),
Category VARCHAR(50),
Brand VARCHAR(50),
Platform VARCHAR(50),
City VARCHAR(50),
Price DECIMAL(10,2),
Quantity INT,
TotalAmount DECIMAL(10,2),
Rating DECIMAL(2,1),
Reviews INT,
OrderDate DATE
);
